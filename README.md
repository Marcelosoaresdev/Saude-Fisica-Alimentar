#  LifeHub – Sistema de Login

Este projeto faz parte do desenvolvimento do **LifeHub**, uma plataforma criada para auxiliar estudantes e usuários no gerenciamento da sua vida acadêmica e pessoal, reunindo recursos como login, IMC, planos e painel do usuário.

##  Sobre o Projeto

Este repositório contém a página de **Login** do LifeHub, desenvolvida com foco em organização, responsividade e usabilidade.  
O objetivo é proporcionar uma experiência simples, clara e eficiente para que o usuário possa acessar a plataforma de forma prática.

##  Tecnologias Utilizadas

- **HTML5** – Estrutura semântica da página  
- **CSS3** – Estilização e layout responsivo  
- **JavaScript (ES6+)** – Funcionalidades do menu e interação do usuário
